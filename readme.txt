WIKIPEDIA SEARCH ENGINE

--------------------------------------------------
Dataset- https://www.dropbox.com/s/0ymmsga6cawgxe1/ire-wiki-search.tar.gz?dl=0

v3_parser.py- parses wiki dump and makes inverted index
v3_file.py- merges index files and split them into smaller chunks
v3_query.py- main query program that returns results in less than 5 seconds
